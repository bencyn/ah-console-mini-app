## AH Console Mini App

A CLI app that interacts with Author's Haven

## Commands

1. View all articles

   `ah list`

2. View specific article

   `ah view <article-slug>`

3. Save specific article to file

   `ah export <article-slug>`

4. View Ah help

   `ah help`
